1. *Automatización*: Capacidad para ejecutar tareas de manera autónoma y programada.

2. *Sensórica*: Uso de sensores para percibir y responder al entorno.

3. *Manipulación*: Capacidad para mover objetos y realizar tareas físicas.

4. *Programabilidad*: Puede ser reprogramado para realizar diferentes tareas.

5. *Inteligencia*: Algunos robots utilizan inteligencia artificial para aprender y tomar decisiones.

6. *Interacción*: Capacidad para interactuar con humanos y otros robots.

7. *Movilidad*: Puede moverse de un lugar a otro según sea necesario.

8. *Adaptabilidad*: Capacidad para adaptarse a cambios en el entorno o en las tareas asignadas.
